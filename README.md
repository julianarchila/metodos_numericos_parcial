# Calibración del modelo Markov-modulado

Proyecto reproducible para el parcial de Métodos Numéricos en Finanzas. El
subyacente asignado es SPY.

## Preparación

```bash
uv sync
uv run jupyter lab
```

Abrir `parcial_calibracion_markov.ipynb`. Todo el desarrollo, las validaciones,
la calibración y las figuras se mantienen en ese único notebook.

## Referencias

- Enunciado: `Julian Alejandro Archila Caro - Parcial Calibracion Markov Modulado.pdf`
- Plan: `plans/plan-parcial-calibracion-markov.md`
- Taller 2: `reference/taller2/Taller_2_Metodos.ipynb`
- Taller 3/4: `reference/taller3/Taller3.ipynb`

## Convenciones

- Los snapshots y datos filtrados se guardan en `data/`.
- Los resultados y figuras regenerables se guardan en `output/`.
- `output/` no se versiona.
